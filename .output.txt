 |
 |  A dropdown control that allows users to select a single option
 |  from a list of [`options`][(c).].
 |
 |  Example:
 |  ```python
 |  ft.Dropdown(
 |      width=220,
 |      value="alice",
 |      options=[
 |          ft.DropdownOption(key="alice", text="Alice"),
 |          ft.DropdownOption(key="bob", text="Bob"),
 |      ],
 |  )
 |  ```
